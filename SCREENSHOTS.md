## Design Elements

### Color Scheme
- **Background**: Dark theme (black/dark gray)
- **Primary Color**: Custom primary color for accents and borders
- **Text**: White/light gray for high contrast
- **Borders**: Subtle borders with hover effects

### Typography
- **Headings**: Bold, black weight fonts
- **Body Text**: Regular weight, readable font sizes
- **Accent Text**: Uppercase tracking for labels and badges

### Interactive Elements
- **Cards**: Hover effects with border color changes
- **Buttons**: Primary color with smooth transitions
- **Upload Area**: Dashed border with hover state
